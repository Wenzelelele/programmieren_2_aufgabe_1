# programmieren_2_aufgabe_1
Übung 1 Programmierübung
Wir haben angefangen, mit der Installation von uv im Browser. Dann haben wir unseren Projektordner angelegt und unser Repository geklont und aus diesem Ordner ein Terminal geöffnet. Im Terminal geben wir dann den kopierten Link vom geklonten Repository ein. Dann haben wir den Befehl uv init ausgeführt, um  das Projekt zu initialisieren. Dann haben wir Numpy importiert über uv add numpy. Das fügt auch die virtuelle Umgebung hinzu. Dann haben wir die Python Dateien geöffnet und einen Bubbblesort Algorithmus hinzugefügt. Im Anschluss haben wir die Sortierten Daten mit Achsenbeschriftungen und skalierter Zeit-Achse geplottet und die Grafik abgespeichert. Das haben wir dann Committed und wieder ins Github gepusht.
